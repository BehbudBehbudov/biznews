# BizNews - News Portal

**BizNews** is a fully functional news portal built with the Django MVT (Model-View-Template) architecture. This application allows users to view news in various categories, write comments, subscribe to newsletters, and send contact information.

## Technologies

- Python 3.x  
- Django 3.x / 4.x  
- HTML / CSS  
- Bootstrap (frontend)  
- SQLite (development) / PostgreSQL (recommended for production)

## Project Structure

### Core Components

- **Models** – Define the database schema  
- **Views** – Handle user requests and return data to templates  
- **Templates** – HTML templates for UI  
- **Forms** – User data input and validation  
- **Context Processors** – Provide global data (e.g., social links, category list)

## News Types

- Main Page Slider (Big Slider)  
- Small Sliders  
- Featured News  
- Popular News  
- Category News  
- Breaking News

## Features

- Publish news in different categories  
- Filter news by category  
- Comment system  
- Newsletter subscription via email  
- Contact form for user messages  
- Pagination for news lists
